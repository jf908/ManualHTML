manualhtml

0.1.1
June 2019

A web manual generator for command-line programs. Source at https://github.com/jf908/ManualHTML.

Examples

manualhtml manualhtml.json

Outputs to man/manualhtml.html

Arguments

-v --version

Output version information and exit.

-o --out

Output directory, defaults to man.

--dark

Output pages in dark mode by default.

--nojs

Excludes any JavaScript from the output.

--verbose

Outputs auxiliary information to the console.
